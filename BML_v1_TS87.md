# Bushido Markup Language — v1.0 (TS87)

| Symbol | Usage | Meaning in Bushido |

| :---- | :---- | :---- |

| `**` | Bold | Truth, pure fact, oath, proof, or core value |

| `*` | Italic | Framing, doubt, whispering, hidden agenda |

| `~~` | Strikethrough | Shed conviction, ego death, old skin |

| `>` | Quote | Reflection from another, ancient wisdom, tradition |

| `:::` | Code block | Ritual, fixed structure, “this is how it should be” |

| `[🔒]` | Label | Confidential/protected – Dojo information |

| `📁 Folder[x]` | Folder link | Reference to project folder |

| `⚠️` | Warning | Active framing signal or energy leak! |

| `✅` | Confirmation | Digital signature or action performed |
